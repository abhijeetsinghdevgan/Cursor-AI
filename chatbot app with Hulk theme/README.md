# GammaChat (Hulk Theme)

This is a simple chat app. You type a message, and Google Gemini replies.

The app has a Hulk/Bruce Banner look:
- Green + purple colors
- Hulk-themed labels and buttons

## What You Need

- A browser (Chrome, Edge, or Firefox)
- A Gemini API key (free from Google AI Studio)

## Get Your Gemini API Key (One Time)

1. Open [Google AI Studio](https://aistudio.google.com/).
2. Sign in with your Google account.
3. Click **Get API key** (or **API keys**).
4. Click **Create API key**.
5. Copy the key.

## Open the App

Choose one option:

### Option A: Download ZIP (easiest)

1. Open this project on GitHub.
2. Click **Code** -> **Download ZIP**.
3. Extract the ZIP.
4. Open the folder `chatbot app with Hulk theme`.
5. Double-click `index.html`.

### Option B: Use Git (for developers)

```bash
git clone git@github.com:abhijeetsinghdevgan/Cursor-AI.git
cd "Cursor-AI/chatbot app with Hulk theme"
```
Then open `index.html` in your browser.

## First-Time Setup in the App

1. Paste your API key into the **Gemini API Key** box.
2. Click **Save Key**.
3. The key panel will hide after saving.

## How to Chat

1. Type your message in the input box.
2. Click **SMASH!!**.
3. While waiting, the button shows **SMASHING...**.
4. Read Gemini's reply in the chat area.

## If You Need to Change Your API Key

1. Click **Change API Key** at the top.
2. Enter the new key.
3. Click **Save Key** again.

## Quick Check (Did Everything Work?)

- App opens in browser without errors.
- You can save your API key.
- You can send a message and get a reply.
- Reloading the page keeps your saved key.

## Simple Security Note

Your API key is stored in your browser on your computer.

Before sharing this project publicly, keep `EMBEDDED_API_KEY` in `script.js` empty.
